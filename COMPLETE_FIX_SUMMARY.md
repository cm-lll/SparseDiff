## 实现细节

### 状态空间映射流程

1. **全局ID → 局部ID**：
   ```
   全局ID 0 → 局部ID 0 (no-edge)
   全局ID offset, offset+1, ... → 局部ID 1, 2, ...
   ```

2. **计算后验分布**（局部状态空间）：
   - 使用关系族特定的转移矩阵
   - 在局部状态空间计算

3. **局部ID → 全局ID**：
   ```
   局部ID 0 → 全局ID 0 (no-edge)
   局部ID 1, 2, ... → 全局ID offset, offset+1, ...
   ```

### 关键优化

- **按batch分组处理**：避免维度不匹配问题
- **局部状态空间计算**：提高准确性，减少计算量
- **正确映射**：确保结果与模型输出兼容
