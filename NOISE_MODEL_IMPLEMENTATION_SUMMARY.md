# 异质图噪声模型实现总结

## ✅ 已完成

### 1. 转移矩阵类 (`HeterogeneousMarginalUniformTransition`) ✅
- 支持关系族隔离的边转移矩阵
- 每个关系族有独立的转移矩阵
- 支持没有子类别的关系族（只有存在性，2 个状态）
- 同质图模式兼容（保留原逻辑）

### 2. 数据集信息 (`ACMSubgraphsInfos`) ✅
- 计算每个关系族的边类型分布（`edge_family_marginals`）
- 存储关系族映射信息（`edge_family2id`, `edge_family_offsets`）
- 存储端点类型信息（`fam_endpoints`）

### 3. 噪声应用 (`apply_sparse_noise`) ✅
- 支持关系族隔离的边扩散
- 已存在边的扩散：按 `edge_family` 分组，使用对应的转移矩阵
- 非存在边的采样（伯努利分布）：为每个关系族独立计算和采样
- 同质图模式保留原逻辑

## 关键实现细节

### 关系族隔离的边扩散流程

1. **已存在边的扩散**：
   - 按 `edge_family` 分组
   - 将全局边属性 ID 转换为关系族内的局部 ID
   - 使用关系族特定的转移矩阵进行扩散
   - 转换回全局 ID

2. **非存在边的采样**：
   - 为每个关系族独立计算 `emerge_prob`
   - 为每个关系族独立进行伯努利采样
   - 使用关系族特定的转移矩阵采样边属性
   - 转换回全局 ID

3. **端点类型考虑**：
   - 从 `fam_endpoints` 获取端点类型信息
   - 当前简化实现：使用全局非存在边数
   - TODO: 更精确的计算需要根据节点类型统计

## 注意事项

1. **没有子类别的关系族**：
   - 只有 2 个状态：`[no-edge, exists]`
   - 为了统一接口，仍然计算分布，但计算量很小

2. **同质图模式兼容**：
   - 当 `heterogeneous=False` 时，自动使用原始逻辑
   - 完全保留原代码的行为

3. **批处理支持**：
   - 所有操作都支持批次维度
   - 关系族隔离在批次内独立处理

## 待优化

1. **非存在边数的精确计算**：
   - 当前使用全局值，应该根据端点类型精确计算
   - 例如：`author_of` 关系族的非存在边 = (Author 节点数) × (Paper 节点数) - 已存在的 `author_of` 边数

2. **非存在边的采样**：
   - 当前使用全局采样，应该根据端点类型约束采样范围
